## 使い方

---

### 以下のコマンドを実行します

```:
cd ${HOME}
curl -sf https://raw.githubusercontent.com/yasudanaoya/dotfiles/master/dotfile.sh | sh -s
```

以上。

#### パターンB

もし、仮に上のがうまくいかなかったらこれで。

```
cd ${HOME}
git pull git@github.com:yasudanaoya/dotfiles.git
cd dotfiles
sh dotfile.sh
```
